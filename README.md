# Memory Puzzle Game (Java)

This is a GUI-based Memory Puzzle Game developed using Java Swing.

Features:
- 4x4 card grid
- Random card shuffle
- Card matching logic
- Interactive GUI

Technologies Used:
- Java
- Swing GUI
- Object-Oriented Programming
- Event Handling

How to Run:
1. Compile: javac MemoryGame.java
2. Run: java MemoryGame
